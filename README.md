# University Exam Registration System (UERS)

A desktop-based **University Exam Registration System** built using **VB.NET (Windows Forms)**. This application allows students to log in, register for exams, and access a simple portal interface. The project is structured with forms for login and portal navigation, along with application configuration and resources.
